Number Guess Game

Game Rules:

Input your guess numbers. Hit the "check it" button for the result.
If it shows "L", it means your guess number is higher than the main number.
- Please enter new number for exact input filed.
If it shows "H", it means your guess number is lower than the main number.
- Please enter new number for exact input filed.
If it shows a solid number, your guess number is right.
- Don't enter new number for the exact filed.
If you guess all of those numbers correctly, you will have a message.
Note: When you refresh your browser, It will generate new number.
